# はじめに

**PaLASOLU付録編 - パーティクルライブのつくりかた**へようこそ！

本ドキュメントは、「パーティクルライブを作ったことがないけれど、はじめてのパーティクルライブ制作に挑戦したい！」という人から、「はじめてのパーティクルライブを完成させたので、次はもっとよいものを作りたい！」という人に向けた、令和最新版のパーティクルライブ指南書を目指しています。

## 対象読者について

本ドキュメントの対象読者は、以下のような人です。
- アバター改変はある程度できるけれど、パーティクルライブを制作したことがない。作ってみたい！
- パーティクルライブ？それって難しいんでしょう？
- パーティクルライブ2作目は、1作目よりもいい感じにしたいなぁ～

逆に、本ドキュメントが対象としていない読者は、以下のような人です。
- 入門編の前提知識に達していない人
	- アバター改変もよくわかっていません！
	- Unityが動かなくなったら、プロジェクトを破壊します。
- パーティクルライブを理解しすぎている人
	- スキルを磨いて商業レベルで戦っていきたいです！
	- ところで、Sanrio Vfesに演出で参加した経験があります。

## 構成について
本ドキュメントは「基礎編」「応用編」「番外編」の3段階で構成されています。

### 基礎編
基礎編の目標は、「はじめてのパーティクルライブをアップロードすること」です。

すなわち、クオリティ面を一切気にせず、**とにかく完成させること**を目標としています。細かい説明や複雑な内容をすべて応用編以降に置くことで、必須な内容のみをできる限り抽出して伝えることをベースとしています。

そのため、一度でもパーティクルライブを作ったことがある人にとってはかなり冗長な内容となることでしょう。そのような人は、応用編から読み始めることをお勧めします。

### 応用編
応用編の目標は、「パーティクルライブを少し良くすること」です。

自分の作ったパーティクルライブに満足がいきませんか？良い作品を作るパーティクルライバーの考えていることを知りたいですか？よいでしょう。応用編では、よりよい作品作りのために、基礎編で触れなかった様々なものに触れていきます。

特に、演出の花形であるところのパーティクルそのものについて深掘りするのは、応用編の重要な内容と言えるでしょう。

### 番外編
番外編の内容は、ここまでの内容とは少し毛色が違います。というのも、パーティクルライブに留まらない範囲の内容を扱うからです。あえて目標を言語化するならば、「感性を理論武装するこお」でしょうか。

それは例えば音の聞き方だったり、色の見え方だったり、文字が及ぼす効果だったり、多岐に渡ると同時に、いわゆる「映像制作」全般に直結するような内容を取り扱います。そのため、抽象性も難易度も一段階上がっています。
必要に応じて応用編と往復しながら、実際に手を動かして納得しながら進むのもよいでしょう。必要な部分だけつまみ食い的に読むのもよいでしょう。

番外編では、ここまでに出てこなかった要素として、Unity以外の様々なツールを活用していきます。最終的な完成物へのイメージを高めるための地力構築として、あらゆる寄り道をしよう。これが、番外編のテーマです。
