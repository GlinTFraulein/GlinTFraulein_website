# Timelineを使おう

:::info[建設中！]
このページはまだ執筆中のため、不完全です！もうちょっと待っててね。画像増やしたりとか、説明丁寧にやったりとか、いろいろやってないの……
:::

ここからは、いよいよ自分のパーティクルライブを作っていきます。

……さて、パーティクルライブのことを思い出してみましょう。楽曲に合わせて様々な3Dオブジェクトやパーティクルが、出たり消えたり、動いたりしていますね。

このような「変化」や「動き」は、パーティクルライブにおいては「Timeline」という機能を使って作られることがほとんどです。逆に、**「Timeline」が使えればパーティクルライブが作れる！**といっても過言ではありません。

本ページでは、Timelineの使い方を学びます。すなわち、目標は以下の2点です。
- パーティクルライブのために、Timelineウィンドウを使う準備をすること
- Timelineにキーフレームを追加し、「変化」や「動き」を作ること



## Timelineをセットアップする
`Tools/PaLASOLU/ParticleLive Setup`

## Timelineのつかいかた

録画してキーを打つ
