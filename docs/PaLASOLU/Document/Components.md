# Components

ここでは、PaLASOLUが提供する各種コンポーネントについて説明します。

[Low-effort Uploader](./LoweffortUploader.md)は個別ページがありますので、そちらをご覧ください。

## PaLASOLU Preset Applier for lilToon

lilToonには「プリセットを適用する」機能があります。本Componentは、この「プリセットを適用する」を、アバター全体に対して非破壊に行うことができるものです。

高度な設定内で、一部分だけ別のプリセットを適用する「例外設定」、プリセットを適用しない「除外設定」をすることも可能です。
